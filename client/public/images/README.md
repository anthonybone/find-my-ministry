# Saint Carlo Acutis Easter Egg Image

## Required Image
Please add a PNG image of Saint Carlo Acutis to:
`/client/public/images/saint-carlo-acutis.png`

## Image Specifications
- **Format**: PNG (for transparency support)
- **Size**: Recommended 64x64 pixels or larger (will be displayed at 64x64)
- **Content**: A nice portrait photo of Saint Carlo Acutis
- **Background**: Preferably transparent or white background

## Suggested Sources
You can find appropriate images from:
- Official Catholic websites
- Saint Carlo Acutis devotional sites
- Catholic media archives
- Vatican news photos

## Fallback
If the image is not found, the easter egg will display "CA" (Carlo Acutis initials) in a blue-purple gradient circle.

## Easter Egg Features
- **Location**: Fixed position in bottom-right corner
- **Animation**: Hover effects with rotation and scaling
- **Tooltip**: Shows "Eucharistic Miracles" on hover
- **Link**: Opens https://www.miracolieucaristici.org/en/liste/list.html
- **Accessibility**: Proper alt text and title attributes

## Implementation Notes
The easter egg is subtle and doesn't interfere with the main UI, but provides a delightful discovery for users who notice it!
